# simple-blender-pipeline
 A lightweight tool for importing blend files. The plugin will automatically take the tscn file generated by godot and unpack it when added to a scene. This allows you to easily recompose the objects of your blend file however you like inside of Godot. The Simple Blender Pipeline also fixes a number of other issues inherent to godot's default pipeline, all while being easy and intuitive to use! There is also a companion addon for blender that you can download from the github page!
